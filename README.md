# sm0k3r

sm0k3r is like [Headcrab](https://github.com/Deadboy666/h3adcr-b) but for Windows.

## How it works

1. Detects your Steam installation and Steam version
2. Checks the version against compatible versions
3. Offers to downgrade if it is too new, offers to upgrade if it is out of date
4. Grabs the SteamTools components and throws them where they go
5. Blocks automatic Steam client updates

## Features

1. Downgrades/upgrades Steam to compatible versions
2. Blocks Steam auto-updates
3. Installs SteamTools without the unnecessary SteamTools app
4. Preserves user data

## Usage

1. Download `sm0k3r.exe` from [Releases](https://github.com/Selectively11/sm0k3r/releases)
2. Run it
3. Select option 1 ("Run everything") to set up everything automatically
